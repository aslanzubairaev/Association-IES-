## Association IES

Этот репозиторий содержит сайт Association IES (Next.js, TypeScript).

### Запуск

Установить зависимости:

- `npm install`

Запуск в режиме разработки:

- `npm run dev`

Сборка:

- `npm run build`

Проверка стиля кода:

- `npm run lint`

### Структура

- `src/app/` — страницы и общий каркас сайта
- `src/components/` — переиспользуемые компоненты интерфейса
- `public/` — статические файлы (иконки/изображения), которые используются в интерфейсе

### Важно про правила проекта

- В каждом файле `.tsx` в начале должен быть комментарий, который простыми словами объясняет, что делает файл.
- Переиспользуемые блоки интерфейса складываем в `src/components/`, а страницы держим максимально простыми.
- Не используем HOC, render props и “кастомные use* функции”.
